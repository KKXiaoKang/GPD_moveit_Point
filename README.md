# GPD_moveit_Point
a simple demo

* you can find how to use gpd in [ROS workspace to get start](https://blog.csdn.net/Eeko_x/article/details/104835154)

## install gpg 
```bash
sudo apt-get install libboost-all-dev


cd gpg
mkdir build && cd build
cmake ..
sudo make install
```

## install gpd rospkg

catkin_make
